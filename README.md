## Setup
Here is a template:
```

<html>
    <head>
        <title>sh0rk</title>
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/722o/catstrap@1.0/catstrapg.min.css" async defer>
    </head>
    <body data-bs-theme="light">
        <h3>Hello, CatStrap! ^w^</h3>
        <p>I'm a Kitty Cat!</p>
    </body>
</html>
```
